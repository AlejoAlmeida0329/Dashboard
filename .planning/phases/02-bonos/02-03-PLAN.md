**Decisión de chart lib (default):** `recharts ^2.15.x`.

Rationale:
- Compatible con React 19 (recharts 2.13+ soporta React 19).
- Bundle ligero comparado con nivo (que carga d3 entero) o visx (que requiere armar primitivas a mano).
- API declarativa SSR-friendly (Recharts renderiza en cliente pero la mayoría de su shell es serializable; lo envolvemos en un componente `'use client'`).
- Patrón conocido en ecosistema shadcn (varios ejemplos oficiales usan recharts).

**Override path (si el ejecutor encuentra problemas):** caer a `nivo` (`@nivo/line`) o a un SVG hand-rolled. Documentar el motivo del override en 02-03-SUMMARY.md como deviation.

**Install:**
```bash
npm install recharts@^2.15.0
```
Confirmar `npm run build` clean post-install (a veces recharts pelea con types — si hay un error TS, agregar `@types/recharts` si existe o reportar).

**Crear `src/components/bonos/BonosChart.tsx`:**
```tsx
"use client";

import { LineChart, Line, XAxis, YAxis, Tooltip, ResponsiveContainer, CartesianGrid } from "recharts";
import type { BonoByDate } from "@/lib/domain/bonos";
import { formatCOP, formatInteger } from "@/lib/format";

type Props = {
  data: BonoByDate[];
  /** Métrica a graficar. Default 'count' (conteo de bonos). 'monto' grafica $ vendido. */
  metric?: "count" | "monto";
};

export function BonosChart({ data, metric = "count" }: Props) {
  // Normaliza para recharts (necesita number, no Date)
  const series = data.map((d) => ({
    date: d.date,
    value: metric === "count" ? d.count : d.monto,
  }));

  const tickFormatter = (n: number) =>
    metric === "count" ? formatInteger(n) : formatCOP(n);

  return (
    <div className="h-[320px] w-full">
      <ResponsiveContainer width="100%" height="100%">
        <LineChart data={series} margin={{ top: 8, right: 16, bottom: 8, left: 16 }}>
          <CartesianGrid strokeDasharray="3 3" className="stroke-border opacity-50" />
          <XAxis dataKey="date" tick={{ fontSize: 12 }} className="text-muted-foreground" />
          <YAxis tickFormatter={tickFormatter} tick={{ fontSize: 12 }} className="text-muted-foreground" />
          <Tooltip
            formatter={(v: number) => tickFormatter(v)}
            labelClassName="text-foreground"
            contentStyle={{ background: "var(--background)", border: "1px solid var(--border)" }}
          />
          <Line
            type="monotone"
            dataKey="value"
            stroke="currentColor"
            strokeWidth={2}
            dot={false}
            activeDot={{ r: 5 }}
          />
        </LineChart>
      </ResponsiveContainer>
    </div>
  );
}
```

**Notas de diseño:**
- "Línea calmada" del 02-CONTEXT.md → `dot={false}`, `strokeWidth={2}`, `type="monotone"` (curva suave).
- Sin colores hardcoded — `stroke="currentColor"` hereda del wrapper. La página puede aplicar `text-primary` o similar al div padre. Esto facilita ajustes de tema.
- Tooltip usa CSS variables del theme — consistencia con shadcn cards.
- Tick formatter pasa por `formatCOP`/`formatInteger` → respeto de Pitfall 9 (single Intl gate).
- Todas las strings de UI en español (consistencia con el resto del proyecto).

NO añadir `data-presenter-hide` al chart — el chart es la pieza HEROÍNA y se muestra en ambas vistas (cliente y interna). El CSS de presenter mode solo afecta los elementos marcados.

```tsx
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card";
import type { BonoByEmpresa } from "@/lib/domain/bonos";
import { formatCOP, formatInteger } from "@/lib/format";

type Props = {
  rows: BonoByEmpresa[];  // ya viene como top10 (truncado por la página)
  rangeLabel?: string;    // texto opcional del período, e.g. "últimos 30 días"
};

export function Leaderboard({ rows, rangeLabel }: Props) {
  return (
    <Card data-presenter-hide>
      <CardHeader>
        <CardTitle className="text-base">Top 10 empresas{rangeLabel ? ` · ${rangeLabel}` : ""}</CardTitle>
      </CardHeader>
      <CardContent>
        {rows.length === 0 ? (
          <p className="text-sm text-muted-foreground">Sin bonos en el período seleccionado.</p>
        ) : (
          <ol className="space-y-2">
            {rows.map((r, idx) => (
              <li key={r.empresa_id} className="flex items-center justify-between gap-3 text-sm">
                <span className="flex items-center gap-2 truncate">
                  <span className="inline-flex h-5 w-5 items-center justify-center rounded-full bg-muted text-xs font-medium">
                    {idx + 1}
                  </span>
                  <span className="truncate">{r.empresa_nombre}</span>
                </span>
                <span className="flex shrink-0 items-center gap-3 tabular-nums">
                  <span className="text-muted-foreground">{formatInteger(r.count)}</span>
                  <span className="font-medium">{formatCOP(r.monto)}</span>
                </span>
              </li>
            ))}
          </ol>
        )}
      </CardContent>
    </Card>
  );
}
```
