    Export 3 functions:

    1. **`deriveEmpresasIndex(transactions: Transaction[], filters: DashboardFilters): EmpresaListRow[]`**
       - Two-pass algorithm. First pass: walk ALL transactions ONCE; for each activity-counting tx, accumulate per-empresa: count, montoHistorico, max(ultimaActividad). Inside the loop, ALSO check if the tx's fecha falls in the filter window (between fromTs/toTs); if yes, accumulate txPeriod, montoPeriod, max(ultimaActividadInPeriod).
       - Empresa name: first-occurrence-wins per empresa_id (mirror of `empresas.ts`).
       - Skip rows with empty/whitespace empresa_id (defensive guard).
       - The empresa filter (`filters.empresa`) is IGNORED here — the table is the place where the user picks an empresa, not where they narrow to one. Document this in JSDoc.
       - status: 'activa' if `ultimaActividadInPeriod !== null`, else 'inactiva'.
       - Sort the output by `montoHistorico` DESC (largest empresas first; matches the Bonos-leaderboard convention).
       - Return new array; do NOT mutate input.

    2. **`summarizeEmpresasIndex(rows: EmpresaListRow[]): EmpresasIndexSummary`**
       - `totalEmpresas = rows.length`
       - `empresasActivas = rows.filter(r => r.status === "activa").length`
       - Empty-input safe: `{ totalEmpresas: 0, empresasActivas: 0 }`

    3. **`findEmpresa(transactions: Transaction[], empresaId: string, filters: DashboardFilters): EmpresaProfileSummary | null`**
       - If empresaId is empty/whitespace, return null.
       - Single pass over transactions. For each activity-counting tx with matching empresa_id, accumulate txHistorico, montoHistorico, ultimaActividad; if also in filter window, accumulate txPeriod, montoPeriod, ultimaActividadInPeriod.
       - Track empresa_nombre via first-occurrence-wins.
       - If no activity-counting tx is ever seen for empresaId, return null (the profile page can render its 404 fallback).
       - status: 'activa' if `ultimaActividadInPeriod !== null`, else 'inactiva'.

    Export 1 function:

    ```ts
    /**
     * 12-month per-empresa activity series for the profile chart (CLI-06).
     *
     * Algorithm:
     *   1. Compute the 12-month window ending at `asOf` (Bogotá month). Default
     *      `asOf = new Date()`. Window includes asOf's month + the prior 11.
     *   2. Filter transactions to: empresa_id === empresaId AND isActivityCounting.
     *   3. Group by Bogotá month (`formatInTimeZone(t.fecha, BOGOTA_TZ, "yyyy-MM")`).
     *   4. Zero-fill: emit ALL 12 months, even if some have zero count/monto.
     *      Without zero-fill, gap months would be invisible — the chart needs
     *      continuous time axis to read "this empresa stopped transacting in X".
     *   5. Sort ascending by month.
     *
     * Empty input or unknown empresa → returns 12 zero-rows for the right window.
     * The chart leaf can read length===12 always and skip empty-state branching.
     */
    export function aggregateMonthlyActivity(
      transactions: Transaction[],
      empresaId: string,
      asOf?: Date,
    ): MonthlyActivity[];
    ```

    Implementation:
    - `BOGOTA_TZ = "America/Bogota"` (file-private const, mirror of recargas.ts:74).
    - Determine the "anchor" month: `formatInTimeZone(asOf ?? new Date(), BOGOTA_TZ, "yyyy-MM")`.
    - Generate the 12 month labels: walk back from anchor by month using a small `subMonthsBogota(label, n)` helper (string-only month arithmetic on `yyyy-MM`). Return labels sorted ASC.
    - Build a `Map<string, MonthlyActivity>` initialized with the 12 labels and zero values.
    - Single pass: for each tx with matching empresa_id and isActivityCounting, compute `m = formatInTimeZone(t.fecha, BOGOTA_TZ, "yyyy-MM")`. If `m` is in the map, increment count and add monto. (Tx outside the 12-month window are silently dropped.)
    - Return `Array.from(map.values())` sorted ASC by month.

    Helper signature:
    ```ts
    /**
     * Subtract `n` calendar months from a `yyyy-MM` label (Bogotá-month).
     * Pure string arithmetic; safe for year boundaries (`2026-01` -1 → `2025-12`).
     */
    function subMonthsLabel(yyyymm: string, n: number): string;
    ```

    Implementation: parse `yyyy` and `mm`, `total = year * 12 + month - 1 - n`, recompose `${Math.floor(total / 12)}-${String((total % 12) + 1).padStart(2, "0")}`.
