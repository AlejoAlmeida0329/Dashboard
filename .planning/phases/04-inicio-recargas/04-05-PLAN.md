    **DeltaBadge.tsx** — Server Component (no `"use client"`). Pure formatting.

    Implementation per 04-RESEARCH.md "Code Examples → DeltaBadge component" (literal code lines 549-585):
    - Props: `{ current: number; prior: number | null }`
    - If `prior === null || !Number.isFinite(prior) || prior === 0`: render `<span className="text-xs text-muted-foreground tabular-nums">—</span>`
    - Else compute `change = (current - prior) / prior` (fraction)
    - Sign: `up` (>0) / `down` (<0) / `flat` (===0)
    - Icon: `ArrowUpRight` / `ArrowDownRight` / `Minus` from `lucide-react`
    - Color: `text-emerald-600 dark:text-emerald-400` / `text-rose-600 dark:text-rose-400` / `text-muted-foreground`
    - Display: `formatPercent(Math.abs(change))` from `@/lib/format` + manual prefix `+` / `−` (Intl doesn't auto-add sign)
    - Wrapper: `<span className="inline-flex items-center gap-1 text-xs font-mono tabular-nums {colorClass}">`

    **KPICardsInicio.tsx** — Server Component. Mirror `bonos/KPICards.tsx` shape.

    Implementation per 04-RESEARCH.md "Code Examples → KPI card with delta" (literal code lines 590-669):
    - Props: `{ summary: InicioDeltaSummary }`
    - Outer div: `grid grid-cols-1 gap-4 sm:grid-cols-2 lg:grid-cols-5`
    - 5 Cards in this exact order:
      1. **GMV** — visible always. `<CardDescription>GMV / Volumen total</CardDescription>` + `<CardTitle className="font-heading text-3xl tabular-nums">{formatCOP(current.gmv)}</CardTitle>` + `<DeltaBadge current={current.gmv} prior={prior?.gmv ?? null} />`
      2. **Comisión** — `data-presenter-hide` on `<Card>`. `formatCOP(current.comision)`.
      3. **Take rate** — `data-presenter-hide` on `<Card>`. `formatPercent(current.takeRate)`.
      4. **Empresas activas** — visible always. `formatInteger(current.empresasActivas)`.
      5. **Bonos vendidos** — visible always. `formatInteger(current.bonosVendidos)`.

    Each card: `<Card><CardHeader><CardDescription>{label}</CardDescription><CardTitle ...>{value}</CardTitle></CardHeader><CardContent><DeltaBadge ... /></CardContent></Card>`.

    Imports: `Card, CardContent, CardDescription, CardHeader, CardTitle` from `@/components/ui/card`; `formatCOP, formatInteger, formatPercent` from `@/lib/format`; type-only `InicioDeltaSummary` from `@/lib/domain/inicio`; `DeltaBadge` from `./DeltaBadge`.

    DO NOT add useSearchParams or any conditional rendering on presenter / empresa state. The visibility is 100% CSS-driven via `data-presenter-hide`.
